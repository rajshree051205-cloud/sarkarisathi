# 🇮🇳 Sarkari Sathi

Sarkari Sathi is a web-based platform designed to help users easily discover and access relevant Indian government schemes based on their category and eligibility. The goal is to simplify awareness and accessibility of ongoing and upcoming government policies.

---

## 🚀 Problem Statement

Many people are unaware of government schemes, eligibility criteria, and benefits available to them. This leads to missed opportunities, especially in rural areas, defense families, farmers, women, and students.

Sarkari Sathi aims to solve this by centralizing and organizing schemes in a simple, user-friendly platform.

---

## 💡 Features

### 🎯 Category-Based Scheme Access
Users can explore schemes based on predefined categories:
- Women
- Army Families / Defence Personnel
- Students
- Farmers
- And other citizen groups

Each category displays:
- Ongoing government schemes
- Upcoming policies
- Newly announced programs

---

### 🔐 User Authentication System
- Login / Signup functionality
- User profile with:
  - Username
  - Email
  - Password authentication
- Secure session-based access (backend in progress)

---

### 🌐 Smart Navigation (Planned/Integrating)
- Direct links to official government portals
- Easy access to apply for schemes
- Simplified navigation for non-technical users

## 🎯 Vision

This ensures that people can actually benefit from policies designed for them.
